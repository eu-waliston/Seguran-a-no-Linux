# Exercícios Práticos

## Desafio 1: Configure um servidor SSH que:

    - Aceita apenas chaves SSH

    - Bloqueia root login

    - Usa porta não padrão

    - Limita tentativas de login

## Desafio 2: Crie um firewall que:

    - Permite apenas tráfego necessário

    - Loga tentativas de acesso bloqueadas

    - Protege contra port scanning

## Desafio 3: Implemente monitoramento que:

    - Detecta tentativas de força bruta

    - Alerta sobre alterações em arquivos críticos

    - Gera relatórios diários de segurança
